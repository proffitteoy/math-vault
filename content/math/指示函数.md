>[!命题]
>任意$R^d$上的可测函数都存在一个单调递增简单函数序列逐点收敛带到$f$，使得$$ \varphi _k(x) \leq \varphi_{k+1}(x) \quad and \quad \lim_{k \to \infty}\varphi_k (x)=f(x)$$

从一个例子开始：$f(x)=x^2$,我们选定$N=2$，让$Q_N$表示以原点为中心的小正方形，这里直接取$N=M=2$,他们分别控制着逼近范围和逼近精度。

按照证明，先构造一个最简单的逼近函数$$F_n=\left\{\begin{array}{cl}
f(x) & \text { if } x \in Q_N \text { and } f(x) \leq N, \\
N & \text { if } x \in Q_N \text { and } f(x)>N, \\
0 & \text { otherwise } .
\end{array}\right.$$
在例子中就是$$\varphi_1 = F_{2,2}=\left\{\begin{array}{cl}
x^2 & \text x \in (-1,1), \\
0 & \text { otherwise } .
\end{array}\right.$$
接下来我们让$N \to \infty$则$\varphi(x) \to f(x)$
取$$E_{l,M}=\{x \in Q_N:\frac{l}{M}<F_N(x)\leq \frac{l+1}{M}\},for 0\leq l\leq NM$$
即$$E_{l,2}=\{x \in (-1,1):\frac{l}{2}<F_N(x)\leq \frac{l+1}{2}\},for 0\leq l\leq 4$$
最后用指数      函数表示$F_{N,M}(x)$$$\sum_l\frac{l}{M}\chi_{E_{l,M}}(x)$$
再严格化证明$$0\,\leq\,F_{N}(x)\,-\,F_{N J M}(x)\,\leq\frac{1}{M}$$对任意的x成立则$$0 \leq F_M(x)-\varphi_k(x)\leq\frac{1}{2^k}$$
